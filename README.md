# xshell-to-Wterminal-color-scheme
small html and javascript converter for xshell color scheme to windows terminal

**note:  it dont work in ie and safari

## befor

    ```
    [terminal.sexy]
    text=c5c8c6
    text(bold)=c5c8c6
    background=1d1f21
    black=282a2e
    black(bold)=373b41
    red=a54242
    red(bold)=cc6666
    green=8c9440
    green(bold)=b5bd68
    yellow=de935f
    yellow(bold)=f0c674
    blue=5f819d
    blue(bold)=81a2be
    magenta=85678f
    magenta(bold)=b294bb
    cyan=5e8d87
    cyan(bold)=8abeb7
    white=707880
    white(bold)=c5c8c6
    [Names]
    name0=terminal.sexy
    count=1
    ```
